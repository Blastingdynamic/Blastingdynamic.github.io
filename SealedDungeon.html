Game Overview

Sealed Dungeon is a third person puzzle game utilizing lasers as the main puzzle element and hazard. The player will utilize lasers to complete puzzles by using mirrors, splitters, and colored gems to properly connect the laser to the goal.

Role, Responsibilities, and Tools

My role in this project was a gameplay programmer. I created the functionality for 2 puzzle elements to add complexity to puzzles, those are the Laser Splitter and the mirror. I utilized Unreal engine 5's blueprint system. Using Interfaces, function libraries and macros to complete my tasks optimally.
