Description

 This is Birds Eye Hunter that I developed with the team Blue Hunters. This game is a Duck Hunt clone where you play as a hunter who is hunting ducks. I am the Project Lead and UI designer of this project.

 This is a game made in unity with a 2D Perspective while incorporating 3D objects. This game features a Highscore system incorporated to record the score of the player when they win or lose the game. The Biggest challenges faced by our team for this were ensuring the crosshair moved according to the mouse and when the player presses the shoot button it would recognize if a Duck was behind it; As well as ensuring that the ducks would move within the bounds of screenspace without the usage of object colliders; and properly recording and displaying score information after a game end.

 In reference to the images on the right are specific parts I developed myself. The first image shows the UI of the game where I display the Round number, number of birds before the round ends, the player's current score, the amount of ammo the player currently has before a strike, and the number of strikes before the game results in a lose condition. The second Image shows the Highscore UI and system I developed to save play scores after a game end condition while being displayed in a highest top ranking order. This system comes with a Clear Score button to wipe the Records just like an arcade system.
